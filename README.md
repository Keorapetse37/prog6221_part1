# Cybersecurity Awareness Chatbot - Part 1

This is a console-based virtual assistant designed to educate South African citizens on cybersecurity. It features a voice greeting, a customized ASCII interface, and an interactive Q&A system covering topics like phishing, passwords, and safe browsing.

## How to Run the Application

### Prerequisites
* **Operating System:** Windows (Required for the `System.Media` audio playback).
* **IDE:** Visual Studio 2022 (or newer).
* **Framework:** .NET 10.0 SDK (or newer).
